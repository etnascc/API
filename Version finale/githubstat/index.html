GitHubStats.



 Sur GitHub, les stats de projet sont souvent utilisées par les différents collaborateurs pour suivre l'évolution du projet, on peut facilement voir qui participe le plus, quelle est la fréquence des commits, observer les ajouts et délétions dans le code etc...

 Le but du défi est d'étendre ces stats à une organisation. Ainsi, il sera possible l'aide d'une application Web de regrouper les statistiques (commits, ajouts, délétions, participations, punch cards) de l'ensemble des projets du groupe, également de pouvoir observer des statistiques concernant les Pull-Requests en cours. Le tout visible sous forme de graphe. GitHub Stats permet une meilleur gestion des étapes de la réalisation et d'organisation d'un projet au sein d'une organisation.
